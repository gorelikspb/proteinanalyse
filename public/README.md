# Sequence Analysis Tools

This folder contains the static website files ready for deployment to Cloudflare Pages.

## Structure

- `index.html` - Main sequence analyzer
- `dna-gc-calculator.html` - GC content calculator
- `reverse-complement.html` - Reverse complement tool
- `orf-finder.html` - ORF finder
- `protein-mw-calculator.html` - Protein molecular weight calculator
- `sequence-translation.html` - DNA translation tool
- `common.js` - Shared JavaScript functions
- `styles.css` - Global styles

## Testing Locally

Open `index.html` in a browser to test. All paths are relative and should work correctly.

## Deployment

See the main `README.md` in the repository root for Cloudflare Pages deployment instructions.
